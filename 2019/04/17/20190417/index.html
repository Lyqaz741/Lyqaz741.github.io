自动化测试

实现用火狐或谷歌浏览器打开百度输入selenium并进行查询

一、按照实际情况下载火狐或者谷歌浏览器对应版本的驱动

二、构建路径

1.打开eclipse创建一个Java项目
2.选择项目中的Referenced Libraries点击右键，选择Bulid Path，点击Configure Build Path
3.根据需要选择对应的jar，点击Libraries选项卡，Add JAR…是添加项目内的包，Add External JARs…是添加项目外的包（这里选择的是项目外的包分别是：selenium-server-standalone-3.11.0.jar和client-combined-3.11.0.jar）
